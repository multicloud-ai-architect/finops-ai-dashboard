## Overview

Multi-cloud FinOps dashboard — Grafana, Opencost, LLM recommendations, AWS + Azure + GCP cost consolidation. This project is part of the MAIA — Multi-Cloud AI Architect portfolio, providing reference architectures and reusable building blocks for cloud architects working across AWS, Azure, and GCP.
